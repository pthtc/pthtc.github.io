argument

  2. 分析错误
  3. 用例子证明

结构

  1. 审题(3min): 找出原文的论证和证据(1.5min)，同时找出逻辑错误和想好论证和例子(1.5min)
  2. 第一段(3-5min): 开头段，归纳论点，说明论点有问题，有逻辑漏洞，准备发起进攻
  3. 第二段-N段(15-20min): 分类别去攻击各个逻辑错误，至少找出3个逻辑问题
  4. 第三段(3-5min):

开头段

CEF

  1. 举出元论证的证据
  2. 指出重要证据
  3. 表明自己对原论证的判断 logical
  4. 指出其存在的逻辑问题

  3. 论证时所用的语言

quality of sample

  • representativeness 代表性 selective/unrepresentative sample 随机性，不够大 性格学历群体家庭背景年龄职业地域
  • meaningfulness/vague data 无意义 张冠李戴
  • vague data 模糊数据 模糊：量词，程度副词，变化幅度，比较级
  • invalid time 时间 模糊就质疑时间，清楚就质疑距离
  • time validity 观测周期要长
  • non-aligned surveyor 非相关性/专业性
  • quantity of the sample 样本数量 只给绝对值/不给相对值/和行业对比/和自我对比

时间模板

In the argument, the observation period is only limited in __. However, we have to doubt the time validity during the test time. In common sense, an observation period should be extensive enough to testify the effeciveness of a certain policy/porduct.

只给绝对值

The argument tells us that “”, However, the auther fails to indicate the percentage of ___. So this evidence is fall too vague to be meaningful.

只给相对值

The argument fails to provide information regarding the absolution number of ___. So this evidence is far too vague to be meaningful.
